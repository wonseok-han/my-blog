# VSCode Recommendations Extensions

해당 프로젝트에서 팀원들의 같은 개발 경험을 위해 VSCode 사용시 확장 플러그인을 권장합니다.

- `astro-build.astro-vscode`: Astro 구성 요소, 스크립트, 스타일 등을 편집할 때 문법 강조, 코드 자동 완성, 오류 감지 등의 기능을 적용하기 위한 확장 플러그인.
- `unifiedjs.vscode-mdx`: MDX 문법 강조, 코드 자동 완성, 오류 감지 등의 기능을 적용하기 위한 확장 플러그인.
